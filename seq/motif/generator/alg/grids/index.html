Grids

Èmilie Gillet, 2013

Grids was a drum pattern generator module for Eurorack, designed by Èmilie Gillet under her brand name, Mutable Instruments. It outputs a pattern for three drums, each playing on a different drum note. Grids has six knobs, thus six parameters:

  • X
  • Y
  • Drum 1 Complexity
  • Drum 2 Complexity
  • Drum 3 Complexity
  • Chaos

X and Y select a drum pattern in a 2D space. This pattern has information for all three drums. The space is smooth, meaning that it is set up so that nearby patterns will tend to sound similar and will morph from one to the other as you move from one pattern to the other in the space.

Each drum note in a drum's pattern has a certain amount of importance to the pattern (lower importance values are more important). If the complexity knob is above the importance of a note, the note will sound, else it will be muted. Thus turning the knob down will allow only "basic" or "critical" notes to be sounded, and turning it up will gradually include more and more embellishments with less important drum notes. This can be done for each of the drums independently (three knobs).

Chaos introduces randomness into whether a drum note is sounded or muted regardless of complexity.

To this Seq adds some additional parameters not found in the original module (as it is based on Gate/CV). First, you can set the MIDI Note which will be output when a drum is sounded. Second, you can set the MIDI Velocity for the same.

Grids optionally accents a drum note if it is very important. If you turn this feature on, you must also stipulate the accent velocity for each drum hit. (Yes, those velocities can be quieter than normal notes if you like).
