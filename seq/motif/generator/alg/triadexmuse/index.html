Triadex Muse

Marvin Minsky and Edward Fredkin, 1969

The Muse is a melody generator which uses two binary counters and a 31-step linear feedback shift register (or LFSR), a kind of random number generator. It produces melodies of notes over two octaves, with or without rests. You can change the volume (velocity), tempo, and pitch of the songs. I have also added the ability to change the length of each note (that is, the gate).

Each timestep, notes are produced by adding together the four intervals. Each interval is either on or off, depending on whether the item it's tuned to is on or off. To determine what the note is:

  1. We start at "Do" (which have set to Middle C unless you changed the transpose value).
  2. If Interval A is ON, we increase the pitch by one semitone.
  3. If Interval B is ON, we increase the pitch by two more semitones.
  4. If Interval C is ON, we increase the pitch by four more semitones.
  5. If Interval D is ON, we increase the pitch further by an octave.
  6. If the Rest switch is turned on, and we are still at "Do" (none of the intervals was ON), then we play a rest (we play nothing).
  7. Otherwise we play the resulting note pitch.
What can the Intervals be tuned to? We can set them to any of (let's assume the first note is note 0):
  • Always ON
  • Always OFF
  • C 1/2. This means ON for the first half of the note, then OFF for the back half.
  • C1. This means OFF for notes 0, 2, 4, 8, ..., and ON for notes 1, 3, 5, 7, ... (changing every note)
  • C2. This means OFF for notes 0, 1, 4, 5, ..., and ON for notes 2, 3, 6, 7, ... (changing every two notes)
  • C4. This means OFF for notes 0...3, ..., and ON for notes 4...7, ... (changing every four notes)
  • C8. This means OFF for notes 0...7, ..., and ON for notes 8...15, ... (changing every eight notes)
  • B1...B31. This means ON for if the LFSR is ON in that slot (1...31), else OFF.
The LFSR is a list of 31 slots, each of which can be set to either ON or OFF. Each timestep we advance the LFSR. This means that the value of each of the slots is moved forward to the next slot. Slot 31's value is set to what slot 30 was. Slot 30's value is set to what slot 29 was, and so on, down to slot 2 being set to slot 1.

What is slot 1's new value then? It is set to a mathematical function of the current values of the four themes. Just like the Intervals, the Themes can be tuned to on, off, C 1/2, C1...8, or B1...B31. But whereas Intervals determine the note, the Themes just determine the next value of LFSR's slot 1. Slot 1 will be set to ON if there are an even number of Themes tuned to ON, and it will be set to OFF otherwise.

Note that the C 1/2 goes On and then Off over the course of one note. This means that to play at the proper rate of one "note" per quarter note, we must have a rate of Eighth Notes by default.

This algorithm has quite illustrious inventors. Marvin Minsky and Edward Fredkin were both quite accomplished people in their own (non-musical) fields. Minsky is considered to be the godfather of Artificial Intelligence, and Fredkin popularized the notion in physics of the universe as a giant discrete computional device.
